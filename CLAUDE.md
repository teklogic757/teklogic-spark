# Claude Code Guidelines for Teklogic Spark AI

## Brand Rules

### Company Name Capitalization
- **Correct:** `Teklogic` (lowercase 'l')
- **Correct:** `TEKLOGIC` (all caps)
- **Incorrect:** `TekLogic` (capital 'L' with mixed case)

The 'L' in Teklogic is NEVER capitalized unless ALL letters are capitalized.

Examples:
- "Welcome to Teklogic Spark AI" (correct)
- "TEKLOGIC INDUSTRIES" (correct)
- "TekLogic Solutions" (WRONG - should be "Teklogic Solutions")
